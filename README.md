## Функционал
Система предназначена для автоматизированного проведения этичных фишинг-симуляций в рамках внутреннего аудита. Поддерживает:
- Загрузку ФИО, email, должность сотрудника через CSV
- Генерацию уникальных фишинг писем для каждого сотрудника
- Массовая рассылка фишинговых писем
- Сбор, аналитика метрик

## Использование
```bash
python3 main.py
```
1. Ввести:
- Почту отправителя
- Пароль от SMPT сервера
- Домен фишингова сайта 
- Если SMPT сервер не gmail.ru, yandex.ru, mail.ru, то ввести:
  - host
  - port
2. Загрузить csv таблицу  
3. Нажать на "Запуск аттаки"
